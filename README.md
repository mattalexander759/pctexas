# Matt's Texas P&C Study Guide: 25-26

This is a static offline/hostable quiz app generated from `tx_pc_master_study_guide_codex_v6_essential.json`.

## Run

Open `index.html` in a browser, or publish this folder as a static site.

No server is required. There are no external dependencies. The app works offline.

## Notes

- Questions are shown in batches of 11.
- This version has 331 questions.
- Missed questions from a batch are reviewed until answered correctly.
- Use **Repeat question** after answering if you got it right but want to see it again.
- Insights are shown after each answer and should be studied.
- The final summary can start a review of the questions missed most often.
- Progress is saved in `localStorage`.
- Use **Reset progress** to clear saved progress.
- Use **Restart current batch** to begin the current batch again.
